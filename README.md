# freeCodeCamp-campers
freeCodeCamp [campers](https://www.freecodecamp.org/challenges/build-a-recipe-box)

Create a recipe box in freeCodeCamp.
- Create a recipe
- View a recipe
- Edit a recipe
- Remove a recipe

---

# Running the app

* Clone the project
```
npm install
npm run start
```
* Go to **localhost:3000** in the browser
